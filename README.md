# Freestyle Career

Simulador de carrera freestyle hip-hop en canvas + TypeScript. El primer MVP es una PWA local: corre en navegador, guarda progreso en `localStorage` y queda preparado para envolverlo como app con Capacitor cuando el loop este validado.

## Desarrollo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## App futura

- Mantener la logica del juego independiente del backend.
- Convertir la build web en app con Capacitor cuando haya MVP estable.
- Agregar backend despues para cuentas, rankings, crews y batallas online.
- Mantener PWA activa para instalacion directa desde navegador.
